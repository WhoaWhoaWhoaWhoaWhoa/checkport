# Управление меню филиалов

React приложение для управления меню филиалов компании.

## Установка и запуск

Перед началом убедитесь, что у вас установлен Node.js версии 16 или выше. Проверить это можно командами: node --version и npm --version.

Установите зависимости проекта: npm install

Запустите приложение в режиме разработки: npm start

Приложение будет доступно по адресу http://localhost:3000

Для создания production сборки выполните: npm run build

Собранное приложение будет находиться в папке build/ и готово к размещению на хостинге.

Структура проекта: 
src/components/ - React компоненты
src/hooks/ - кастомные хуки
src/services/ - API взаимодействие  
src/styles/ - стили
src/utils/ - утилиты
src/App.js - главный компонент
src/index.js - точка входа

Приложение использует API по адресу: https://testjob.checkport.ru
